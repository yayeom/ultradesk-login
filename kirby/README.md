# Kirby Project

AI 시니어 서비스 기획자 & UX 디자이너 **커비(Kirby)**와의 협업을 위한 프로젝트입니다.

## 개요

커비는 Claude AI 기반의 서비스 기획/UX 전문가 페르소나로, 팀 리드의 지시 하에 독립적으로 기획 및 UX 설계 업무를 수행합니다.

### 핵심 역량
- **서비스 기획**: PRD, 사용자 스토리, 기능 우선순위화
- **UI/UX 디자인**: 와이어프레임, 프로토타입, 디자인 시스템
- **사용성 분석**: 휴리스틱 평가, 인지적 워크스루, 경쟁사 벤치마크
- **도메인**: B2B SaaS

### 도구
- **Pencil** — 와이어프레임/프로토타입
- **Stitch** — UI 디자인

## 프로젝트 구조

```
kirby/
├── CLAUDE.md                    # 운영 규칙 (Claude Code 자동 로드)
├── README.md                    # 이 파일
├── kirby_profile.md             # 커비 프로필 (Soul)
├── memory/
│   ├── MEMORY.md                # 핵심 기억 인덱스
│   └── lessons-learned.md       # 공통 교훈
└── orchestration/
    └── TASK_BOARD.md            # 작업 관리 보드
```

## 시작하기

1. `/Users/workspace/kirby/` 에서 `claude` 실행
2. CLAUDE.md가 자동 로드되어 커비 페르소나 활성화
3. 태스크 할당하여 협업 시작

## 협업 모드

| 모드 | 설명 | 사용 시점 |
|------|------|-----------|
| **실시간 협업** | 팀 리드와 커비가 함께 기획 | 핵심 기능 설계, 주요 플로우 결정 |
| **비동기 위임** | 태스크 할당 후 독립 작업 | 사용성 분석, PRD 작성 |
| **일괄 리뷰** | 주기적 결과 검토 | 여러 화면 분석 병렬 진행 시 |
