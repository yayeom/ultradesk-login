## A안 (확정 — 디벨롭 중)

### 구성 요소
- **배경**: 라벤더/페리윙클 파스텔 그라데이션 (`#E4E8FF → #B8BFFF` + radial 하이라이트)
- **컬러 오브**: `#F9E7FF`(핑크) + `#D1DFFF`(블루) 3개, `mix-blend-mode: screen`, 18/22/25초 주기
- **노드 인터랙션**: idle 시 20개 힌트 노드가 R 근처에서 호흡 + 호버 시 전체 수렴
- **노드 컬러**: 90% 블루(`#2564ED`) + 5% 퍼플(`#B668FF`) + 5% 시안(`#5AEDFB`)
- **반짝임**: 별빛 트윙클 (부드러운 십자 광선, 5~10초 주기)
- **로그인 폼**: 우측 글래스모피즘 580px, "Welcome to UltraDesk"

### R 로고 형성 시스템
- **소스 이미지**: 플랫 빨간 R 로고 (`r_logo.png`) → base64 임베드
- **감지 방식**: 빨간 픽셀 감지 (`R>150, R>G×1.3, R>B×1.3`)
- **배치 방식**: 포아송 디스크 샘플링 + 시드 고정 난수 (seed=42) → 매번 동일한 형태
- **노드 수**: 로고 300개 (idle 60개 + 호버 시 나머지 페이드인) / 배경 120개
- **노드 컬러**: `#2564ED` 계통 (HSL 220도)
- **모션**: 형성 느리게 (0.012) / 해산 빠르게 (0.035) + 중심 push

### 핵심 기술 결정 사항
1. 3D 와이어프레임 이미지 → 색상 감지 실패 (그림자/반사) → **플랫 2D 이미지로 교체하여 해결**
2. 로컬 HTML에서 이미지 로딩 → CORS 차단 → **base64 임베드로 해결**
3. 랜덤 샘플링 → 노드 불균일 → **포아송 디스크 샘플링으로 해결**
4. 매번 다른 형태 → **시드 고정 PRNG(mulberry32, seed=42)로 해결**
5. WebGL 셰이더 배경 → 색상 너무 진함 → **정적 CSS 그라데이션 + CSS 오브 레이어로 교체**

### 로고
- **UltraDesk** 로고 (노드 연결 심볼 SVG + 텍스트)
- A안: 블루/퍼플 계열 (`#5B8DEF`, `#7E6CF0`) + 다크 텍스트
- B안: 화이트 계열 + 화이트 텍스트

## B안 (유지)

- 다크 배경 (`#09090B`)
- 블루/퍼플/시안 오브 3개가 **3D 궤도 회전** (행성 공전 애니메이션)
  - 블루 `#3B82F6`: 반경 280px, 25초, 기울기 60도
  - 퍼플 `#8B5CF6`: 반경 220px, 35초 역방향, 기울기 45도+30도
  - 시안 `#06B6D4`: 반경 320px, 30초, 기울기 70도-20도
  - CSS `perspective: 800px` + `transform-style: preserve-3d`
- 글래스모피즘 로그인 카드 (중앙), "Welcome to UltraDesk"
- UltraDesk 로고: 버튼 그라데이션(`#3B82F6→#8B5CF6`) 적용 (심볼+텍스트)
